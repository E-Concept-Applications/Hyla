
Afficher des informations sur un fichier ou répertoire (lancement de plugin d'affichage : dir ou gallery)

	Afficher des infos et les commentaires sur un fichier :
		query string :		http://localhost/list/?aff=info,/toto.zip
		path info :			http://localhost/list/index.php/info/toto.zip
		mod rewrite :		http://localhost/list/toto.zip?info

	Afficher des infos sur un répertoire (listage) :
		query string:		http://localhost/list/?aff=info-10,/dir
		path info :			http://localhost/list/index.php/info-10/dir		-> Commence l'affichage à partir de la 10 ième occurence
		mod rewrite:		http://localhost/list/dir?info

	Afficher des infos sur un fichier contenu dans une archive :
		query string:		http://localhost/list/?aff=info,/toto.zip!file.txt
		path info :			http://localhost/list/index.php/info/toto.zip!file.txt
		mod rewrite:		http://localhost/list/toto.zip!file.txt?info



Téléchargement de fichiers :

	Télécharger un fichier :
		query string:		http://localhost/list/?aff=download,/toto.zip
		path info :			http://localhost/list/index.php/download/toto.zip
		mod rewrite:		http://localhost/list/toto.zip?download

	Télécharger un fichier zip contenant les fichiers d'un répertoire :
		query string:		http://localhost/list/?aff=download,/dir
		path info :			http://localhost/list/index.php/download/dir
		mod rewrite:		http://localhost/list/dir?download

	Télécharger un fichier contenu dans une archive :
		query string:		http://localhost/list/?aff=download,/toto.zip!file.txt
		path info :			http://localhost/list/index.php/download/toto.zip!file.txt
		mod rewrite:		http://localhost/list/toto.zip!file.txt

	Afficher une photo du cache :
		query string:		http://localhost/list/?aff=mini|500,/dir/photo.jpg
		path info :			http://localhost/list/index.php/mini-500/dir/photo.jpg
		mod rewrite:		http://localhost/list/toto.zip!file.txt


Edition

	Editer un fichier :
		query string:		http://localhost/list/?aff=edit,/toto.zip
		path info :			http://localhost/list/index.php/edit/toto.zip
		mod rewrite:		http://localhost/list/toto.zip?edit

	Editer un répertoire :
		query string:		http://localhost/list/?aff=edit,/dir
		path info :			http://localhost/list/index.php/edit/dir
		mod rewrite:		http://localhost/list/dir?edit
