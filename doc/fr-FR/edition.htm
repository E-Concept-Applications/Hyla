Introduction

Hyla vous permet de pouvoir éditer un objet courant (qu'il soit fichier ou dossier) simplement afin de lui attribuer une description, un plugin, une icone...

Bien sûr, pour pouvoir éditer un objet, il faut posséder les droits adéquats.

Pour les fichiers et dossiers

Description

Il est possible d'assigner une description à n'importe quel fichier ou dossier, ainsi, vous pourrez par exemple, expliquer à vos visiteurs l'historique du fichier, les conditions de la prise de vue pour une photo, bref, ce que vous souhaitez !

Uniquement pour les dossiers

Plugins

Lorsque vous éditez un dossier, il est possible de modifier son plugin par défaut (voir document sur le fonctionnement de Hyla.)

Des cases à cocher vous permettent de choisir le plugin à lancer automatiquement à l'ouverture du dossier courant, ainsi, il vous est possible de spécifier le plugin "Gallery" qui vous créera automatiquement une galerie d'images.

Icones

Il est possible d'attribuer une image à un dossier, permettant, par exemple, d'avoir une idée du contenu grâce à une image représentative.

Droits

De la même manière que dans l'administration, il est possible de gérer les droits de l'objet courant directement depuis l'édition de ce dernier.

Vous pourrez constater que si le dossier courant ne possède pas de droit, Hyla cherche automatiquement le dossier parent en possédant.

Si le dossier courant possède des droits, Hyla vous propose alors de les éditer ou de les supprimer par le biais du lien ou des cases à cocher correspondantes.

Remarque : L'édition des droits est uniquement visible et accessible pour les administrateurs.
