Introduction

La page information est disponible pour tout type d'objet et vous permet d'obtenir des informations à propos de l'objet courant.

Cette page est divisée en trois catégories : Général, Syndication et Exportation.

Général

Cette catégorie vous permet d'obtenir des informations comme la taille de l'objet, son type, et éventuellement calculer, si c'est un fichier ou un fichier archivé, son md5.

Des liens vous permettent de télécharger l'objet courant en différent format d'archive.

Syndication

La syndication vous permet d'être tenu au courant de l'activité du site à l'aide des flux rss.

Pour cela, il vous faut utiliser un lecteur de flux rss, notons que Mozilla Firefox et Mozilla Thunderbird sont capables de lire de tel flux.

Exportation

L'exportation permet comme son nom l'indique d'exporter le contenu d'un plugin afin de l'afficher hors de Hyla.
Cette fonction est très utile pour les blogs, forums souhaitant afficher au sein de leurs messages des objets tel que des documents csv, des vidéos, le contenu d'archives...

Exporter en javascript, ajax

Pour exporter le contenu d'un plugin dans un élément html en javascript à l'aide de jQuery, procédez ainsi :

  1. Copiez le contenu du champ texte
  2. Insérez le code ainsi copié

Téléchargez jQuery sur le site officiel, copiez la librairie sur votre site et ajoutez le code html suivant afin de charger jQuery : 

	
<script language="javascript" type="text/javascript" src="/chemin_vers/jquery.js"></script>

Puis, ajoutez le code ci-dessous : 

	   
   <script type="text/javascript">
       $("#div_name").load("http://localhost/hyla/index.php/obj-export-content/");
   </script>

Le contenu du plugin sera alors inséré dans l'élément possédant l'id "div_name" (ex: <div id="div_name" ...).

Exportation en iframe

Pour exporter le contenu d'un plugin dans une iframe sur votre site, suivez les étapes suivantes :

  1. Sur la page exportation, copiez le contenu du champ de texte "Exportation en iframe".
  2. Ensuite, collez simplement le code html dans le billet de votre blog ou autre

Une iframe contenant le plugin exporté sera alors visible sur votre site.
