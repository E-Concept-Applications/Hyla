Dans cette version de Hyla (0.8.1), il n'y a pas de migration depuis la version 0.8.0 car la structure des tables n'a pas évoluée.
Si vous utilisez encore la version 0.7.2, vous devez d'abord migrer vos données grâce à l'installeur de la version 0.8.0.

Standard

Cette installation consiste simplement à lancer le script install.php se trouvant à la racine du dossier d'Hyla et de suivre les différentes étapes.

Voici les différentes étapes de l'installeur automatique de Hyla

  • 1. Bienvenue dans l'installeur d'Hyla !
  • 2. Vérification de la configuration du serveur
  • 3. Type d'installation

Si vous choisissez l'installation normale :

  • 4. Saisi des informations
  • 5. Création du fichier de configuration
  • 6. Création des tables et insertions des données
  • 7. Création de l'utilisateur principal (Saisi)
  • 8. Création de l'utilisateur principal

Installation manuelle

Bien que ce ne soit pas recommandé (n'oubliez pas de changer le mot de passe admin par la suite), vous pouvez aussi installer Hyla de façon entièrement manuelle, voici la marche à suivre :

  1. Copiez le fichier /conf/config.inc.default.php en /conf/config.inc.php et éditez le correctement (tout est expliqué dans le fichier)
  2. Exécutez (par exemple, avec phpMyAdmin) le script SQL d'installation dans /src/db/mysql.sql
  3. Vous pouvez éventuellement éditer le fichier /conf/hyla.ini
  4. Créez un fichier vide nommé lock dans /conf/

L'utilisateur créé par défaut par cette voie est 'admin' avec le mot de passe 'hyla', une fois loggué dans l'administration, changez le mot de passe immédiatement !
