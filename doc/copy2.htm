/ toto / tata /

Copie

Dans cette exemple, on désire copier les fichiers contenu dans le sélecteur du dossier /toto/tata/, ils sont indiqué dans la barre de l'objet courant.
On a l'information sur la dernière sélection multiple de fichiers et cela permet ainsi de bien savoir sur quoi va porter l'action courante.
Le sélecteur permet ainsi de changer sa sélection avant l'action.
